# 🌸 Meihua Robot - ABU Robocon 2026
This repository contains the ROS 2 Humble workspace for the **Meihua** autonomous robot. This design is specifically engineered for the "Kung Fu Quest" challenge, featuring a 6-wheeled mechanism for stair climbing and stable navigation.

## 🛠️ Project Structure
* **src/meihua_description**: Contains the URDF model, meshes, and textures for the robot.

## 🚀 Lab Setup Instructions (USP Labs)
To quickly set up a fresh lab computer with all the necessary tools, open your terminal and run:

run install
